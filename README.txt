# saucedemo
 BDD for Saucedemo website  using gherking

Sauce Demo Test Suite

Full regressions
To execute the full test suite, simply run:
$ mvn test

And each scenario will execute
Prerequisites

You need to have maven, java, and Chrome installed on your computer to run

Feature	Scenario						End to End
login	Verify that a valid user can sign in			No
login	Verify that an in-valid user cannot sign in		No
orders	Buy the maximum and the minimum price item of the store	yes

Example: Execute regression tests 

$ mvn test
